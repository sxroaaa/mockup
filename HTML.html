MANUAL HTML

HTML es el lenguaje informatico principal con el que se crean y se estructuran las paginas web. las paginas son vistas desde navegadores por lo tanto HTML es el lenguaje que usan los navegadores para mostrar las paginas webs al usuario. se utiliza para especificar si el contenido de la pagina se debe tomar como un titulo, un parrafo, una lista, un formulario, imagen, etc.

Como surgio HTML

El origen de HTML se remonta a 1980, cuando el físico Tim Berners-Lee, trabajador del CERN (Organización Europea para la Investigación Nuclear) propuso un nuevo sistema de "hipertexto" para compartir documentos.
